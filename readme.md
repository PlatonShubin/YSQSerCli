Описание:

	Базовый функционал клиента. Происходит подключение к серверу. Данные приходят потоково через активный сокет и преобразуются в блок данных.
	Блок данных обрабатывается ядром, если предназначен ядру, или одним из расширений, если предназначен им.
	Для отправки данных серверу формируются и затем отправляются блоки данных.
	
	Базовый функционал сервера. Сервер ждёт клиентов, слушая порт. При подключении клиента сервер запоминает его сокет и далее работает с ним.
	Данные приходят потоково через активный сокет и преобразуются, минуя блок данных, в блок команды.
	Блок команды либа обрабатывается ядром, либо одним из расширений.
	
	Расширения. Нестатические классы, работающие с клиентом/сервером.
